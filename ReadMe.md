# 🧠 Learning Laravel from Scratch – Step-by-Step

> I'm not jumping concepts. This is my structured journey from absolute beginner to confident Laravel developer.

## 🎯 Goal

Learn Laravel the right way — one small concept at a time, with real-world analogies and challenges.

## 📚 Course Roadmap (Instructor-led)

| Phase | Topic |
|-------|-------|
| 1 | What is a framework? (Laravel = IKEA for PHP) |
| 2 | Client, Server, HTTP (how browsers talk to code) |
| 3 | Installing PHP, Composer, and Laravel |
| 4 | MVC pattern: Model, View, Controller |
| 5 | Routing (URLs → code) |
| 6 | Controllers (the brain) |
| 7 | Blade templates (dynamic HTML) |
| 8 | Migrations (database version control) |
| 9 | Eloquent ORM (database = PHP objects) |
| 10 | Forms, CSRF, validation |
| 11 | Eloquent relationships |
| 12 | Authentication (login/register) |
| 13 | Middleware |
| 14 | File uploads |
| 15 | API basics |

*More steps added as I progress.*
